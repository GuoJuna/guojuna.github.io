# Markdown介绍

Markdown 是一种轻量级标记语言,创始人为约翰*格鲁伯(John Gruber). 它允许人们"使用易读易平的纯文本格式编写文档,然后转换成有效的 XHTML(或者HTML)文档".这种语言吸收了很多电子邮件中已有的纯文本标记的特性.

# 标题

行首插入 1 到 6 个 # ，对应到标题 1 到 6 阶

# 一级标题

# 二级标题

# 三级标题

# 四级标题

# 五级标题
# 六级标题

注: # 和 [一级标题] 之间建议保留一个字符的空格,这是最标准的Markdown的写法

# 列表

Markdown 支持有序列表和无序列表。

无序列表使用星号、加号或是减号作为列表标记：

# 无序列表

- 减号1
- 减号2
- 减号3

+ 加号1
+ 加号2
+ 加号3

* 星号1
* 星号2
* 星号3
  • 减号1
  • 减号2
  • 减号3
  • 加号1
  • 加号2
  • 加号3
  • 星号1
  • 星号2
  • 星号3

# 有序列表

1. 文本1
2. 文本2
3. 文本3

2. 文本1
4. 文本2
3. 文本3

有序列表标记上使用的数字并不会影响输出的结果

  1. 文本1

  2. 文本2

  3. 文本3

  4. 文本1

  5. 文本2

  6. 文本3

混合:

1. First
2. Second:
	* Fee
	* Fie
	* Foe
3. Third
  1. First
  2. Second:
    • Fee
    • Fie
    • Foe
  3. Third

注: - 和 1. 与文本之间要保留一个字符的空格

# 链接与图片

在 Markdown 中，插入链接不需要其他按钮，你只需要使用 [显示文本](链接地址) 这样的语法即可，例如：

Autolink: <https://example.com>

Link: [Example](https://example.com)

Reference style [link][1].

[1]: https://example.com  "Example"

注：插入图片的语法和链接的语法很像，只是前面多了一个 ！。

Autolink: https://example.com (opens new window)

Link: Example (opens new window)

Reference style link (opens new window).

这是一个链接 (opens new window)

# 引用

在我们写作的时候经常需要引用他人的文字，这个时候引用这个格式就很有必要了，在 Markdown 中，你只需要在你希望引用的文字前面加上> 就好了，例如：

markdown 引用

注：> 和文本之间要保留一个字符的空格。

# 粗体

Markdown 的粗体和斜体也非常简单，用两个* 包含一段文本就是粗体的语法，用一个* 包含一段文本就是斜体的语法。例如：

 *一盏灯*， 一片昏黄；**一简书**， 一杯淡茶。 守着那一份淡定， 品读属于自己的寂寞。 保持淡定， 才能欣赏到最美丽

最终显示的就是下文，其中「一盏灯」是斜体，「一简书」是粗体：

     this is Java code space
  *```

    this is Java code space
